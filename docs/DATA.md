# Setting Up Data Paths

The instructions on how to create symlinks for ImageNet and CIFAR are given below.

Expected datasets structure for ImageNet or your own datasets:

```
imagenet
|_ train
|  |_ n01440764
|  |_ ...
|  |_ n15075141
|_ val
|  |_ n01440764
|  |_ ...
|  |_ n15075141
```
**or**
```
imagenet
|_ train
|  |_ n01440764
|  |_ ...
|  |_ n15075141
|_ val
|  |_ n01440764
|  |_ ...
|  |_ n15075141
|_ train.txt
|_ val.txt

```
